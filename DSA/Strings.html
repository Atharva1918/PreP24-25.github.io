Strings

What is an Strings ?

String is basically an object that represents sequence of char values. An array of characters works same as Java string. For example:
char[] ch={'P','r','e','P','2','4','2','5'};
String s=new String(ch);
is same as:
String s="Prep24-25";

String class provides a lot of methods to perform operations on strings such as compare(), concat(), equals(), split(), length(), replace(), compareTo(), intern(), substring() etc.

The Java String is immutable which means it cannot be changed. Whenever we change any string, a new instance is created. For mutable strings, you can use StringBuffer and StringBuilder classes.
